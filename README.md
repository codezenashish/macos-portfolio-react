# macOS Portfolio

A modern portfolio website built with React, styled to look like macOS interface.

## 🚧 Work in Progress

This project is currently under development.

## 🛠️ Tech Stack

- **React** 19.2.0
- **Vite** - Build tool
- **Tailwind CSS** 4.1.17
- **GSAP** - Animations
- **Lucide React** - Icons
- **Day.js** - Date utilities

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/codezenashish/macos-portfolio-react.git

# Navigate to project directory
cd macos-portfolio

# Install dependencies
bun install
# or
npm install
```

## 🚀 Development

```bash
# Start development server
bun dev
# or
npm run dev
```

## 📝 Available Scripts

- `bun dev` - Start development server
- `bun build` - Build for production
- `bun preview` - Preview production build
- `bun lint` - Run ESLint
